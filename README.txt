This file only describes the environment and execution step of the 3 main tasks, if you want to know those of bonus, please read the readme.txt in Bonus/Video and Bonus/Blur respectively.

we use Google Colab to develop and execute or code. If you want to run on local machine, please refer to requirements.txt.



2. Execution
Execute the code:

bash run.sh <path/to/data>
ex.
bash run.sh ./data_0611/data/testing/

Finally, a directory named final_outdata will be generated, and 3 directories are included.
